# Quick Certificate Print Guide

## 🖨️ How to Print a Certificate

### Step 1: Open Certificate
- Open certificate in web browser
- Verify all content displays correctly
- Ensure certificate loads completely

### Step 2: Open Print Dialog
**Windows/Linux**: Press `Ctrl+P`
**Mac**: Press `Cmd+P`

### Step 3: Configure Print Settings

| Setting | Value |
|---------|-------|
| **Orientation** | Landscape |
| **Paper Size** | A4 |
| **Margins** | None / 0mm |
| **Scale** | 100% |
| **Background Graphics** | ✓ ON |
| **Headers/Footers** | OFF |

### Step 4: Choose Output
**Option A: Print to Printer**
1. Select your printer
2. Choose paper quality (Fine/High)
3. Click Print

**Option B: Save as PDF** (Recommended)
1. Select "Save as PDF"
2. Name: `Certificate_[Name]_[Date].pdf`
3. Click Save

### Step 5: Verify Output
✅ Certificate fits on single page
✅ All text visible
✅ Signature section shows 2 columns
✅ Colors are vibrant

## 📋 Print Checklist

### Before Printing
- [ ] Open certificate page
- [ ] Check all information is correct
- [ ] View in landscape orientation
- [ ] Verify no content is cut off

### Print Settings
- [ ] Set orientation to **Landscape**
- [ ] Set paper size to **A4**
- [ ] Set margins to **None/0mm**
- [ ] Scale set to **100%**
- [ ] Background graphics **ON**

### After Printing
- [ ] Allow ink to dry (2-3 minutes)
- [ ] Inspect for quality issues
- [ ] Store flat or in protector sleeve
- [ ] Consider lamination for protection

---

## 🎯 Quick Troubleshooting

| Problem | Solution |
|---------|----------|
| Content cut off | Set margins to 0mm, use PDF export |
| Text too small | Increase scale to 110-120%, or use PDF viewer zoom |
| Colors washed out | Enable "Background graphics", check printer settings |
| Signatures on separate page | Ensure A4 landscape orientation, 0mm margins |
| Looks different at home | Export to PDF first for consistency |

## 📐 Paper Recommendations

- **Type**: Certificate or premium paper
- **Weight**: 200-300 gsm (heavier = better feel)
- **Finish**: Matte (avoids gloss reflections)
- **Brightness**: 90+ (whiter, more professional)

**Popular Brands**:
- Southworth Premium
- Crane's Ecru
- Mohawk Fine Papers

## 🌐 Browser-Specific Tips

### Chrome / Edge
```
1. Ctrl+P (or Cmd+P on Mac)
2. More settings → Orientation: Landscape
3. Paper size: A4
4. Margins: None
5. Background graphics: ON
```

### Firefox
```
1. Ctrl+P (or Cmd+P on Mac)
2. Orientation: Landscape
3. Paper format: A4
4. Margins: 0mm all sides
5. Print backgrounds: ON
```

### Safari
```
1. Cmd+P
2. More options → Orientation: Landscape
3. Paper size: A4
4. Show details → check all settings
```

## 💾 PDF Export Method (Best)

1. Press `Ctrl+P` (or `Cmd+P`)
2. Select **"Save as PDF"** from printer dropdown
3. **Filename**: `Certificate_[Name].pdf`
4. Click **Save**
5. Use PDF viewer to print later
6. Ensures consistent output across devices

---

## 🎨 Ink Saving Tips

- Use **High Quality** setting (not Best)
- Print in **Color** mode (not grayscale)
- Consider **Matte finish** paper (less ink)
- Test on regular paper first
