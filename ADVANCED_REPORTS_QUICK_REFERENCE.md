## 📍 Where to Find Reports

1. Log in as **Admin** or **Manager**
2. Navigate to **LMS Analytics & Learning Reports** (from admin menu)
3. Select report from the **categorized sidebar**
4. Apply **department filter** (if needed)
5. **Export** as Excel, PDF, or CSV

- **Struggling Learner?** → Check "Detailed Learner Analytics" for low progress/hours

### Managing Courses
- **Course Not Popular?** → Check "Course Analytics" for enrollment and completion rates

### Managing Teams

- **Admin Workload?** → Check "Admin User Activity" for task distribution

### Compliance & Certificates
- **Track Credentials?** → Check "Compliance & Certificates" for issued certificates

## 📊 Export Your Reports

### How to Export
1. Generate report (automatically loads data)
2. Click export button:
   - 🟢 **Excel** - Download as .xlsx spreadsheet
   - 🔴 **PDF** - Download as formatted PDF
   - 🔵 **CSV** - Download as comma-separated values

### File Naming
All exports automatically named: `{report-name}-report-{DATE}.{format}`

Example: `learner-details-report-2026-04-05.xlsx`

## 🎯 Key Metrics Explained

| Metric | Meaning | Good Value |
|--------|---------|------------|
| **Progress %** | How much of course completed (0-100%) | >80% |
| **Completion Rate %** | % of students who finished course | >70% |
| **Learning Hours** | Total time invested in learning | >40h/quarter |
| **Engagement Level** | High (>100h), Medium (20-100h), Low (<20h) | High |
| **Assessment Score** | Average performance on tests | >75% |
| **Streak (Days)** | Consecutive days active | Longer = Better |
| **Interaction Rate %** | % of content interacted with | >80% |
